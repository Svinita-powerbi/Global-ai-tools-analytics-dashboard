📊 Project Summary : Global AI Tools Analytics Dashboard

The Global AI Tools Analytics Report is an interactive multi-page Power BI dashboard designed to analyze the global AI tools ecosystem across categories, pricing models, user ratings, API availability, and market growth trends.

This project provides comprehensive insights into AI tools performance, adoption patterns, and pricing strategies using advanced data modeling and DAX calculations.

🎯 Project Objectives

Analyze AI tools by category and subcategory

Track monthly traffic and active users

Compare pricing models (Free, Freemium, Paid, API-based)

Evaluate user ratings and agentic capability scores

Identify top-performing and lowest-performing tools

Monitor yearly growth and release trends

📈 Key Insights

Conversational & Image Generation tools dominate traffic.

Freemium pricing model is most popular.

Tools released after 2020 show rapid growth.

API-enabled tools have strong enterprise adoption.
